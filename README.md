
# Heavy Auto Health - CARB Compliance Testing Services

Mobile CARB compliance testing services for heavy-duty vehicles in California.

## Features

- Mobile testing service
- Online appointment scheduling
- Real-time pricing calculator
- Secure payment processing
- Customer portal
- CARB compliance guidance

## Installation

1. Clone the repository:
```bash
git clone https://github.com/jasonkong0608/heavyautohealth.git
cd heavyautohealth
```

2. Install dependencies:
```bash
npm install
```

3. Configure environment variables:
- Copy `.env.example` to `.env`
- Update Square API keys and other configuration

## Development Setup

1. Start the development server:
```bash
npm run dev
```

2. Access the site at `http://localhost:3000`

### Development Requirements

- Node.js 16+
- npm or yarn
- Square developer account

## Deployment

1. Build the project:
```bash
npm run build
```

2. Deploy to production:
```bash
npm run deploy
```

The site is automatically deployed to GitHub Pages when pushing to the main branch.
